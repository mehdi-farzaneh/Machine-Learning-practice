
## Machine Learning Practice

This repository contains MATLAB/Octave example implementations for core machine learning algorithms (exercise-style):

- **K-Means Clustering/** — scripts and data for K-means clustering.
- **Linear Regression/** — single/multiple variable linear regression and regularization examples.
- **Logistic Regression/** — binary and multiclass logistic regression, regularized variants.
- **Neural Networks/** — neural network forward/backprop and multiclass examples.
- **Support Vector Machine/** — SVM training and examples.

Usage
 - Open the project in MATLAB or GNU Octave.
 - Run the main example script in a folder (e.g., `KMeans_Clustering.m`, `LinearReg.m`, `LogisticReg.m`, `NN.m`, `SVM.m`).
 - Some examples use `fmincg.m` or other helper functions included in the folders.

Notes
 - Data files (`.mat`, `.txt`) are included per folder.
 - Designed for MATLAB; GNU Octave should work for most scripts but minor adjustments may be needed.


Source: Final exam of the Machine Learning Course, represented by Dr. Andrew Ng at Stanford University via Coursera.
